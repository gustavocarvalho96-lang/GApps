Como usar:
1. Abra o arquivo "Abrir App Portatil.bat".
2. O lancador tenta abrir em uma janela de app pelo Edge ou Chrome.
3. Se preferir, abra diretamente "ReceitasMedicas_Portatil.html" no navegador.
4. Para levar em pendrive, copie a pasta inteira mantendo estes arquivos juntos:
   - ReceitasMedicas_Portatil.html
   - styles.css
   - app-data.js
   - app.js
   - Abrir App Portatil.bat

O app roda localmente, sem instalacao e sem internet.

Atalhos:
- Ctrl + K: focar busca
- Ctrl + Enter: copiar o protocolo atual
- Esc na busca: limpar busca

Recursos praticos:
- Favoritos ficam salvos no navegador usado para abrir o app.
- Em receitas, e possivel copiar tudo, so a receita ou so as orientacoes.
