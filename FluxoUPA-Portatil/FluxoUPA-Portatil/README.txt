Fluxo UPA Portatil

Como abrir:
1. Entre nesta pasta.
2. De dois cliques em abrir_app.bat.
3. Se preferir, abra index.html direto no navegador.

Como levar para outro computador:
Copie a pasta FluxoUPA-Portatil inteira para um pendrive ou compacte em .zip.

Observacao importante:
O app nao precisa instalar nada, mas a leitura automatica de imagem usa bibliotecas carregadas da internet
na primeira abertura do navegador: html2canvas e tesseract.js. A entrada manual funciona pelo proprio HTML.

Se quiser uma versao .exe totalmente offline, sera necessario instalar Node/npm ou usar outro computador com
essas ferramentas para gerar um pacote Electron/Neutralino.
